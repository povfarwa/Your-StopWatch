# Minimalist Stopwatch

A distraction-free, full-screen stopwatch website designed with an elegant, ultra-dark color palette (`#1A1A1D` and `#3B1C32`). 

Unlike traditional containerized layouts, this stopwatch is built to scale naturally across the entire browser viewport, focusing purely on essential typography and a clean user experience.

## Features

* **Full-Screen Presentation:** No restrictive boxes or containers. The interface utilizes fluid typography that dynamically scales to fit any screen size perfectly.
* **Custom Muted Palette:** Built utilizing a deep, dark gray base (`#1A1A1D`) paired with rich plum accent borders (`#3B1C32`).
* **High-Precision Timing:** Tracks milliseconds with visually distinct, low-contrast text elements so you can focus on the active seconds.
* **Smart Lap Tracking:** The lap window stays hidden out of sight, sliding smoothly into view only when the first lap is actively recorded.
